# CLIO Kit AI DevKit

This directory contains development resources, documentation, and reference materials for building MCP servers and agent tools within the CLIO Kit (part of the IoWarp platform).

## Contents

### FastMCP Documentation
- **`FASTMCP_2.0_DOCUMENTATION.md`** - Comprehensive FastMCP 2.0 patterns, decorators, and migration guide
- **`FASTMCP_LATEST_FEATURES.md`** - Latest FastMCP features and best practices for CLIO Kit MCP servers

### Design System
- **`design_principles/DESIGN.md`** - IoWarp platform design system (neobrutalist principles) for CLIO Kit website and UI components

### h5py Documentation
- **`h5py_docs/`** - Reference documentation for h5py library, used by HDF5 MCP server

## Purpose

This devkit provides:
- **Reference Documentation**: FastMCP patterns, h5py usage, design guidelines
- **Migration Guides**: Upgrading existing MCP servers to latest FastMCP features
- **Best Practices**: Code patterns, performance optimization, error handling
- **Design Resources**: Visual identity, color system, component patterns

## Usage

These documents are reference materials for:
- **MCP Server Developers**: Building new servers or upgrading existing ones
- **Contributors**: Understanding code patterns and design principles
- **Maintainers**: Ensuring consistency across CLIO Kit components
